SearchX — Built to Prove a Point

I built SearchX because I kept telling clients what great eCommerce search looks like. Now I can show them.

SearchX is a full-stack AI-powered eCommerce search platform running on a local Kubernetes cluster. It indexes over 34,000 real products and serves them through a hybrid BM25 and vector search pipeline that combines lexical precision with semantic understanding — the same approach I've recommended to retailers for years, now running as a live demo I own end to end.

The platform includes a React UI with home, search, and category pages; ECK Elasticsearch for search infrastructure; ArgoCD for GitOps-based deployment; Prometheus and Grafana for full observability; and a local RAG pipeline powered by Ollama with Gemma 3. The Java embedding layer uses Deep Java Library and PyTorch with no Python runtime dependency — a deliberate architectural choice that keeps the stack clean and enterprise-friendly.
